Information Collection and Use

The Habit Me app does not collect any personal information from users. The app operates completely offline and stores user input, such as habit names, actions, and notification times, into a local database. This information is solely used within the app to schedule notifications and display user records.

Data Storage and Security

User input is stored locally on the user's device using Android's internal app contextual storage and Room as the database. This ensures that user data remains secure and is only accessible by the app itself. Liquid Stars (Yehor Tkachuk) takes measures to ensure the security of user data within the app.

No Third-Party Services or Ads

The Habit Me app does not use any third-party services or display advertisements. User data is not shared with any third parties.

Data Management

Users have the option to clear app data or remove the app entirely, which will delete all locally stored data associated with the app.

Age Restrictions

There are no age restrictions for users of the Habit Me app.
